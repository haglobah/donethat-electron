# Contributing

This repository is public primarily for transparency and inspection of the DoneThat desktop client.

We are not currently operating this repository as a general open contribution project.

## Current Contribution Posture

- Issues for desktop-client bugs are welcome.
- Security reports should follow [SECURITY.md](SECURITY.md), not public issues.
- Proprietary backend internals are out of scope for this repository.
- General unsolicited pull requests are not guaranteed review or acceptance.

## Changes We May Consider

We may review small, well-scoped pull requests when they:

- clearly affect only the desktop client in this repository
- align with the public roadmap at `https://donethat.ai/roadmap`
- match the existing coding style and keep changes focused
- include tests or verification steps when behavior changes
- align with current maintainer priorities

Large refactors, speculative rewrites, and backend-facing protocol changes are unlikely to be accepted.

Before investing significant work in a pull request, open an issue first so we can confirm it is in scope.
